# 💝 Valentine's Day React Project

Ek romantic aur interactive Valentine's Day website React + Vite mein banayi gayi hai!

## ✨ Features

- 🎨 Beautiful pink gradient design
- 🧸 Animated teddy bear
- 💕 Floating hearts background
- 🎯 Interactive YES/NO buttons
- 🎉 Confetti celebration
- 📱 Fully responsive
- ⚡ Fast aur smooth animations

## 🚀 Installation & Setup

### Step 1: Dependencies Install Karo
```bash
npm install
```

### Step 2: Development Server Run Karo
```bash
npm run dev
```

Ab browser mein `http://localhost:5173` pe jao!

## 📦 Build for Production

```bash
npm run build
```

Build files `dist/` folder mein mil jayengi.

## 🎮 How It Works

1. **YES Button**: Click karne par celebration screen aayegi with confetti! 🎊
2. **NO Button**: 
   - Har click pe button move hota hai
   - YES button bada hota hai
   - NO button chota hota hai
   - Messages change hote hain

## 🛠️ Tech Stack

- ⚛️ React 18
- ⚡ Vite
- 🎨 CSS3 Animations
- 💖 Pure CSS (No extra libraries)
